# Path Reference

This is the current layout used by Argo CD ApplicationSets and CI automation.

## Helm Sources

- Shared microservice chart:
  - `platform-helm/general/microservices`
- Per-service base values inside chart:
  - `platform-helm/general/microservices/<service>/values.yaml`

## Environment Values

- Dev microservice overlays:
  - `platform-helm/envs/dev/applications/microservices/<service>-values.yaml`
- Prod microservice overlays:
  - `platform-helm/envs/prod/applications/microservices/<service>-values.yaml`
- Dependency values:
  - `platform-helm/envs/<env>/applications/app-dependencies/databases/<name>/values.yaml`
  - `platform-helm/envs/<env>/applications/app-dependencies/message-queue/kafka/values.yaml`
- Platform component values:
  - `platform-helm/envs/<env>/platform/<domain>/<component>/values.yaml`

## Preview Values

- PR overlays generated by CI:
  - `platform-helm/envs/preview/microservices/preview-<PR_NUMBER>/<service>-values.yaml`

## GitOps ApplicationSets

- Dev:
  - `platform-gitops/env-appsets/dev/platform-apps.yaml`
  - `platform-gitops/env-appsets/dev/dependency-apps.yaml`
  - `platform-gitops/env-appsets/dev/microservices.yaml`
  - `platform-gitops/env-appsets/dev/policy-apps.yaml`
- Prod:
  - `platform-gitops/env-appsets/prod/platform-apps.yaml`
  - `platform-gitops/env-appsets/prod/dependency-apps.yaml`
  - `platform-gitops/env-appsets/prod/microservices.yaml`
  - `platform-gitops/env-appsets/prod/policy-apps.yaml`
- Preview:
  - `platform-gitops/env-appsets/preview/preview-appset.yaml`

## CI Workflows That Touch Values

- Preview create/update:
  - `.github/workflows/ci-tasks/update-gitops.yaml`
- Preview cleanup:
  - `.github/workflows/preview-envs/preview-destroy.yaml`
