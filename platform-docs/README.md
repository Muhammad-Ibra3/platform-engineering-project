# Platform Documentation

This directory holds platform-level documentation for engineers working in this repository.

## Structure

- `architecture/` - System architecture, boundaries, and deployment flows.
- `decision-log/` - Architectural decisions and trade-offs.
- `developer-guide/` - Step-by-step guides for common engineering tasks.

## Start Here

- Platform architecture: `architecture/README.md`.
- New to this repo: read `developer-guide/README.md`.
- Adding or changing services: read `developer-guide/add-microservice.md`.
- Updating platform or dependency charts: read `developer-guide/update-platform-and-dependencies.md`.
- Working with previews: read `developer-guide/preview-environments.md`.
